CSS Motion Path Demos

Pure Tailwind utility classes. No custom CSS required.

1. Circle Loader 🟢

offset-path-[circle(80px_at_100px_100px)] animate-motion

2. Rectangle 🟦

offset-path-[xywh(20px_40px_160px_120px_round_20px)] animate-motion-ping

3. SVG Path 〰️

offset-path-[path('M_20,100_C_60,20_140,20_180,100')] animate-motion-bounce

4. Infinity ♾️

offset-path-[path('M_100,100_C_..._Z')] animate-motion

5. Heart ❤️

offset-path-[path('M_100,60_C_..._Z')] animate-motion-alternate

6. Car on Road 🚗

🚗

7. Rocket 🚀
